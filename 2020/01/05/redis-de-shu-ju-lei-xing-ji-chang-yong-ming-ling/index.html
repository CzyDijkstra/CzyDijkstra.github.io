redis的数据类型及常用命令

自己在学习中的一些整理，非直接引用：
redis常用的五大数据类型：String、List、Set、Hash、Zset

key的常用命令

keys * 查看当前所有key
flushall/flushdb 删除所有数据/删除当前数据库数据
select num 切换到数据库num，从0开始到15
move key db 把某个key移除
exists key 判断某个key是否存在
expire key NUM 为给定的key设置过期时间NUM秒
ttl key 看看还要多久过期 -1：永不过期 -2：已经过期，直接移除内存系统
type key 查看你的key什么类型
get/set key (value)得到/设置key，set时如果已经有了，会覆盖key
dump key序列化key

  • 常见数据类型操作命令 redisdoc.com
  • redis的回应，起效1，不起效0

String —最基本的类型

特性和属性

  • 一个key对应一个value
  • 二进制安全，可以包含任何数据，比如序列化对线或者jpg图片
  • value最多512M(理论数据)
  • 单值单value

常用命令

  • set/get/del/append/strlen 和java类似
  • Incr/decr/lncrby/decrby 自增/减 加减数字 一定是要数字才能使用
  • mset/mget/msetnx 多数据set/get,msetnx若有数据不存在，全部失效
  • getrange/setrange
  • setex(set with expire)/setnx(set if not exist)

Hash哈希 类似java里的Map<String,Object>

特性和属性

  • 键值对结合，KV模式不变，但是V是一个键值对
  • 一个String类型的映射表

常用命令

  • hset/hget 单个，注意特性一
  • hmset/hmget 批量
  • hgetall key 获得key中全部值
  • hdel key key1 把key中对应的key1删除
  • hlen
  • hexits myhash v1 查询hash中v1的key是否存在
  • hkeys/hvals myhash 把myhash中全部的key/value显示出来
  • hincrby/hincrbyfloat key key1 num 使key中key1的值+num，整数/小数
  • hsetnx key key1 value1 判断是否存在，若存在则添加

List列表 底层实际是个链表

特性和属性

  • 按照插入顺序排序，可以插入一个元素到列表的头部或者尾部
  • 单值多value
  • 字符串链表，left、right都可以插入或者添加
  • 如果键不存在，创建新的链表，如果已经存在，新增内容
  • 如果值全部移除，对应的键也就消失了
  • 对头和尾操作效率很高，操作中间元素效率惨淡

常用命令

  • lpush/rpush mylist 1 2 3 4 5 把1-5push到list（l=left，r=right）
    rpush是怎么进怎么出，lpush反之
  • lrange mylist a b 把list中a-b范围内的东西显示出来，0 -1 显示全部
  • lpop/rpop mylist 移除一个对应栈顶的
  • lindex mylist num 查看list中第num+1个元素，从上到下
  • llen
  • lrem mylist a b 在list中删除a个b
  • ltrim mylist index_begin index_end截取指定范围的值后再赋值给key
  • rpoplpush list01 list02 把list01的最后一个移至list02的头

*
如：list01 1 2 3
list02 4 5 6
执行后—> list01 1 2 ；list02 3 4 5 6
*

  • lset list01 a b 把索引a处的元素值改为b
  • linsert mylist before/after a b在值为a的元素前/后插入元素b

Set集合

特性和属性

  • 底层是HashMap
  • 无序，无重复，通过HashTable实现
  • 单值多value

常用命令

  • sadd set01 1 1 2 3 向set01中加入1，2，3并且会直接去重
  • smembers set01 查看集合的全部成员
  • sismember set01 a 判断set01中是否存在a
  • scard set01 获取集合里元素个数
  • srem key value 把key中的value删除
  • srandmember key num 在key中随机出num个数字
  • spop key 随机出栈
  • smove key1 key2 num 把key1的num赋给key2
  • 数学集合类
    • 差集 sdiff key1 key2(key1-key2)
    • 交集 sinter key1 key2
    • 并集 sunion key1 key2

Zset 有序集合

特性和属性

  • 跳跃排序
  • 通过分数为集合中的成员进行从小到大的排序。每个元素都会关联一个double类型的分数
  • 成员唯一，但是分数可以重复

常用命令

  • zadd zset01 60 v1 70 v2 80 v3 两个值一个整体
  • zrange key a b (withscores)
  • zrangebyscore key a b 显示分数a-b的key
    • withscores
    • (a/(b 指不包含，如zrangebyscore zset01 (20 30
    • limit num1 num2 从num1开始截取num2个元素
  • zrem key v1 删除key中的某个vlaue
  • zcard/zcount key 统计个数
    • 后面+ a b 代表分数区间
  • zrank/zscore key score/values 获得下标值或者分数
  • zrevrank key values 逆序获得下标值
  • zrevrange key a b 逆序获得a-b内的值
  • zrevrangebyscore key a b
